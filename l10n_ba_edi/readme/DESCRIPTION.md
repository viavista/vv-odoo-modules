Elektronska fiskalizacija (e-fiskalizacija) za Bosnu i Hercegovinu.

Sadrži:

- Upravljanje fiskalnim uređajima i konfiguracija
- Tok fiskalizacije faktura (čekanje, štampano, greška)
- Praćenje fiskalnih brojeva računa (VBR/SBR)
- X-izvještaj i Z-izvještaj (dnevni zaključak)
- Revizijski dnevnik fiskalnih operacija
- Podrška za podijeljeno plaćanje (gotovina, kartica, virman, ček)
- REST API za integraciju eksternih fiskalnih uređaja
