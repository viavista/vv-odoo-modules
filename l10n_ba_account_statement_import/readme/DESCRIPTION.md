Import bank statement files from Bosnian and Herzegovinian banks into Odoo.

Supported formats:

- **XML** - Proprietary XML format used by Raiffeisen BA and other BiH banks
- **MT940** - SWIFT MT940 format with BiH-specific conventions

This module extends the OCA bank statement import framework
(`account_statement_import_file`) with parsers for BiH bank formats.
